## Decisions Made

- **2-commit split (feat + docs) over single-commit:** Task 1 (Python harness) is the load-bearing behavior change; Task 2 (bash header + canaries.md) is pure documentation/registration. Splitting lets `git bisect` pinpoint harness-body regressions independently of canaries.md drift, and matches the Phase 3.1 Plan 01/02/03/04/05 atomic-per-task commit pattern.
- **Phase 2 via sed-copy mock (not SKIP-by-default):** plan originally could have left Phase 2 as a TODO with nightly-only execution. Chose to execute Phase 2 in the default CI path (with SKIP fallback) because the Issue #1 regression surface is the sole reason Phase 2 exists — ensuring it actually runs in smoke mode (not just when `claude_agent_sdk` is present) is the whole value. Plan 02 Task 2 Test E already proved the sed-copy strategy works on this host; Canary #7 reuses the same 3-replacement pattern.
- **stdlib-only reaffirmed under pressure:** Phase 2 tempted a `psutil.Process(pid).ppid()` call for the PPID check. Used `subprocess.check_output(["ps", "-o", "ppid=", "-p", PID])` with `ValueError` / `CalledProcessError` guards instead. Zero new deps; matches D-VERIFY-TOPO-01 Rationale ("합성 canary 가 목적에 충분").
- **`_pid_alive` treats EPERM as alive:** matches the Bash `kill -0 $PID` convention documented in engine-invocation-protocol.md §Liveness probe. A permission error on the signal send means the PID exists but is owned by another UID — unreachable in this harness (we own the PID we spawned), but the convention keeps the helper reusable for future watchdogs.
- **PASS stdout grammar extended (not replaced):** Plan 01 Task 3 placeholder had `PASS: canary #7 (subagent orphan survived ${DURATION}s, PPID=1)`. Phase 2 required a new label, so appended `; real-chain integration: <PASS|SKIP (reason)>` to the existing grammar rather than breaking it. Acceptance criterion substring match (`PASS: canary #7 (subagent orphan survived 10s, PPID=1; real-chain integration: `) passes; Plan 07 lint rows that scan for the substring continue to work.
